# Kkong - Basic HTML Website

A personal portfolio website built with HTML5 and CSS3. This project features a clean, modern, and responsive interface, focusing on a comfortable user experience. and focus on using correct HTML tags and structure

## Goal

* **Multi-pages:** Consists of Home, Projects, Articles, and Contact pages.
* **Modern Contact Form:** The contact page uses an elegant, centered "Card" style design (like the login page).
* **Sticky Footer:** The footer will always be at the very bottom of the screen, even if the page content is small.
* **Social Media Integration:** Interactive social media icons using FontAwesome in the footer.
* **Responsive Layout:** Using Flexbox for neat element layout across screen sizes.

## Requirements
* Pages: Home, Project, Article, dan Contact
* Navigation bar with links to each page, at the top of each page
* Structure the content in a way that makes adding styles easy
* Add SEO meta tags
